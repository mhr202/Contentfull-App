# Contentful App
## Local Development (Mac OS / Ubuntu)

 - Ruby version 2.7.1

**System Level Dependencies:**

 - PostgreSQL 12.4

**Project Level Dependencies**

 - bundle install

**Setting up the Database**

 - rails db:create 
 - rails db:migrate 
 - rails db:seed

**Running the Rails Server**

 - rails server

**Running Tests**

- bundle exec rspec
